# Fulfillment Orchestrator (WIP)

Claude Agent SDK orchestrator for a logistics workflow: ticket triage,
shipment document processing, order fulfillment.

## Status
Early days. Summarizer/validator/enrichment/fulfillment agents wired up
through a central orchestrator. Document pipeline handles the sequential
5-stage flow.

## Run locally
```
pip install -r requirements.txt
python scripts/run_local.py
```

More docs to come once the architecture settles down.
